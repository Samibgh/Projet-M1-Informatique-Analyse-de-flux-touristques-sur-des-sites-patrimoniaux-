Projet intégré du Master Informatique


Bienvenue sur le site réalisé par des étudiants de la formation Master Informatique de l'Université Lumière Lyon 2. Il à été créé lors d'un projet dans le but de réstituer des données récuppérés sur un site internet ou une API.

Afin d'obtenir des résultats visuels sur un site internet, le projet s'est déroulé en plusieurs tâches distinctes. Tout d'abord, nous devions choisir deux ou plusieurs site issus du patrimoine mondial de l'UNESCO afin de les comparer. Nous avons pour cela choisis la Tour de Pise et le Sanctuaire d'Itsukushima. Ensuite il nous fallait déterminer sur qu'elle plateforme nous allions recuppérer les données et aussi par quel moyens. Nous avons décidés de scraper les données sur le célèbre site TripAdvisor grâce à la librairie "Puppeteer" de NodeJs. Les valeur séléctionnées sur le site ont été choisit judicieusement afin que par la suite nous ayons de quoi les mettre sous forme de flux. Grâce à cette méthode nous avons pu récolter environ 2600 avis. Lorsque le scraping fut terminé, nous avons importer les données sur Python afin de les nettoyer, les mettre en forme et de créer d'autres variables nous aidant pour l'analyse. Après, nous avons mis les données dans une base de données de PHPMYADMIN pour faciliter la manipulation de celles-ci lors de la création du site. En effet, nous nous sommes connectés avec du code PHP à la base de données et avons réalisées quelques requêtes pour nous permettre d'obtenir des résultats. Ces résultats ont donc été utilisés pour la création de graphiques à l'aide de la librairie Chart.js de JavaScript et d'une carte à l'aide de la libraire Am4charts de JavaScript.

Le projet s'est déroulé de septembre jusqu'à début décembre. Nous avons beaucoup aimés ce projet car il nous a permit de nous entrainer à travailler en équipe en répartissant les différentes tâches. Nous avons pus découvrir les techniques de récupération de données qui sont bien utilisés dans le monde du travail comme par exemple pour le metier de Data Scientist. Cela nous a permit également de nous améliorer sur les différents langagues tels que Java Script et PHP et découvrir de nouvelles librairies utiles et intéressantes.

Nous remercions nos enseignants Mme Bentayeb et M Boussaid pour nous avoir guidés et donné de bons conseils tout au long de ce projet.
